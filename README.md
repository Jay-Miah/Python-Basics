# 🐍 Python Fundamentals Project  
*A Data Technician Bootcamp Project*

This project demonstrates my ability to use **core Python programming concepts** to build simple applications that perform calculations, make decisions, and interact with users. It highlights the foundational logic skills developed during my Data Technician bootcamp.

---

## 🚀 Project Overview
The goal of this project was to write small Python programs that collect user input, process data, and produce meaningful outputs. Through a series of exercises, I learned how to structure logic, control program flow, and handle different data types — all essential skills for real‑world programming and data work.

---

## 🧠 Key Skills Demonstrated

### 🔤 Variables, Data Types & Type Casting
- Creating and updating **variables** to store user data  
- Converting between types using `int()`, `float()`, and `str()`  
- Understanding how Python handles numbers, strings, and booleans  

These skills allowed me to perform calculations and ensure user input was processed correctly.

---

### 🖨️ User Interaction with `print()` and `input()`
- Displaying messages and results using `print()`  
- Collecting user responses with `input()`  
- Building interactive scripts that respond to user choices  

This formed the basis for creating simple, user‑friendly programs.

---

### 🔀 Conditional Logic with `if` Statements
- Writing `if`, `elif`, and `else` blocks  
- Comparing values and making decisions  
- Creating branching logic to handle different scenarios  

This enabled the program to behave differently depending on user input.

---

### 🔁 Loops: `for` and `while`
- Using `for` loops to repeat actions over sequences  
- Using `while` loops to run logic until a condition changes  
- Building counters, menus, and repeated calculations  

Loops helped automate repetitive tasks and build more dynamic programs.

---

## 🧩 How These Skills Came Together
Throughout the project, I combined these fundamentals to:

- Build simple calculators  
- Validate user input  
- Create menu‑driven programs  
- Automate repeated tasks  
- Produce clear, readable output  

Each script demonstrated how basic programming constructs work together to form complete, logical solutions.

---

## 🎯 Key Outcomes
- Gained confidence writing Python code from scratch  
- Learned how to structure logic and control program flow  
- Built interactive programs using input and output  
- Strengthened foundational skills for future data and automation projects
